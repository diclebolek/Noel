# Images Klasörü

Bu klasöre yılbaşı ağacı için kullanılacak resim dosyalarınızı ekleyebilirsiniz.

## Örnek Resim Dosyaları

- `star.png` - Yıldız resmi
- `ornament-red.png` - Kırmızı süs topu
- `ornament-blue.png` - Mavi süs topu
- `ornament-yellow.png` - Sarı süs topu
- `envelope.png` - Zarf resmi
- `tree-ornament.png` - Ağaç süsü
- `gift.png` - Hediye kutusu
- `bow.png` - Kurdele

## Kodda Kullanım

Resimleri HTML'de şu şekilde kullanabilirsiniz:

```html
<img src="images/star.png" alt="Yıldız" class="custom-image">
```

veya CSS'de:

```css
.custom-element {
    background-image: url('images/ornament-red.png');
    background-size: contain;
    background-repeat: no-repeat;
}
```
